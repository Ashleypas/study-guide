HTML

  • HTML : Hypertext Markup Language
  • The head element contains information about the webpage.
  • Meta data : data about webpage that the browser uses to effectively serve the webpage to users
  • Title element definds the title of the webpage
  • Style element defines the css style associated
  • Base element defines the base url for webpage
  • Link element connects an external resource to the HTML document
  • Meta element definds metadata This declaration tells the browser what document type it should expect when it is opened
  • script element embeds client-side scripts in the HTML document
  • lang="" : language of webpage
  • !doctype : This declaration tells the browser what document type it should expect when it is opened
  • Body element defines the body of the document, which contains the content that is visible to the user.
  • Body element is an example of a semantic HTML element
  • Semantic HTML element the name of the element describes its content and the role it plays on a webpage
  • h1 element creates a heading
  • p element creates a paragraph
  • ul,ol and li elements repersents unorder list , ordered list, and list item
  • img element contains information about images that are displayed on the webpage
  • src attribute is the most important because it defines the location of the image
  • br creates a line of empty space
  • a element stands for anchor. the anchor creates a link to other webpages or link to on same webpage
  • href attribute points to the URL for the link
  • section elements are semantic elements that define a section in a document that contains thematic content

Css

  • Css: Cascading Style Sheets
  • margin indicates how much space we want around the outside of an element
  • padding indicates how much space we want around the content inside of an element
  • inline css: apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
  • internal css style sheet: contains CSS rules for the webpage in the head section of the HTML file.
  • exturnal css style sheet: keeps all CSS rules in a separate file
  • Declarations: contain two important components: the CSS property we want to apply and the value of the property
  • class attribute: allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
